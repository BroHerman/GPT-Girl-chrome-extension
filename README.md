# Add an animated GPT-Girl to ChatGPT web UI

### Usage

1. Download and unzip `gpt-girl-extension.zip`
2. Open `chrome://extensions/`.
3. Turn on **Developer mode**.
4. Click **Load unpacked** and select the unzipped folder `gpt-girl-extension\`.
5. Open `https://chatgpt.com/`. GPT-Girl will appear.
6. Use bottons at bottom-right to turn ON/OFF the GPT-Girl and music.

# 在网页版ChatGPT的中加入一个会动的女孩

### 使用方式

1. 下载并解压`gpt-girl-extension.zip`
2. 打开 `chrome://extensions/`。
3. 开启右上角 **开发者模式**。
4. 点击 **加载已解压的扩展程序**，选择解压后的文件夹`gpt-girl-extension/`。
5. 打开 `https://chatgpt.com/` 开始使用，GPT 小女孩会自动出现。
6. 使用左下角的按钮来开启/关闭女孩和音乐。
